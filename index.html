Quick Help

Steps to get your The Blue Alliance API key and to open a Google Sheet for scouting.

Get TheBlueAlliance API Key
(required to fetch matches & rankings)
  1. Open a browser and visit: https://www.thebluealliance.com/account
  2. Sign in or create an account if needed.
  3. Under "API Keys" click "Create New Key" (or copy an existing one).
  4. Paste that key into the "TBA API Key" field in System Setup and Save.
Open a Google Sheet (Scouting)
(optional: paste id & API key into Scouting fields)
  1. Open Google Sheets at https://sheets.google.com and create or open a spreadsheet.
  2. From the sheet URL copy the spreadsheet ID (the long string between /d/ and /edit).
  3. Get a Google API key (enable Sheets API in Google Cloud Console) and paste here.
  4. Save settings. The app will fetch the "Sheet1" range and use row 1 as headers (column A should be team number).

Tip: If you don't want to enable the Sheets API, you can leave the scouting fields empty. The app will still show matches using the TBA key.
